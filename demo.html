📚 Progress Tracker Implementation Guide

Welcome! This guide will walk you through implementing the book progress tracker step by step. By the end, you'll have a beautiful visual timeline showing your writing journey from Prewriting to Published.

Step 1: Understanding the Data Structure

STEP 1

First, you need to add progress data to each book object. Here's what each book needs:

{
    title: "Your Book Title",
    // ... other book data ...
    
    // NEW FIELDS FOR PROGRESS TRACKER:
    currentStage: "Drafting",        // Which stage you're in
    stageProgress: 30,               // Progress % in current stage (0-100)
    wordCount: 45000,                // Current word count
    targetWordCount: 150000,         // Goal word count
    published: false                 // true when published
}
💡 Note: The currentStage must be one of these exact values:
  • "Prewriting"
  • "Drafting"
  • "Revising"
  • "Editing"
  • "Publishing"
  • "Published" (only when published = true)

Step 2: Add the CSS Styles

STEP 2

Copy all the CSS from the /* BOOK STATUS TRACKER */ section in your stylesheet. This includes:

  • .book-status-tracker - Main container
  • .status-timeline - The horizontal line
  • .status-progress-bar - Animated fill bar
  • .status-step - Each stage circle
  • .status-icon - The circular icons
  • Animation keyframes for pulse-active
⚠️ Important: Make sure you copy ALL the CSS, including the different states:
  • .status-step.completed
  • .status-step.active
  • .status-step.published

Step 3: Create the Generator Function

STEP 3

Inside your renderBooks() function, add this helper function:

// Define the 6 stages
const stages = ['Prewriting', 'Drafting', 'Revising', 
                'Editing', 'Publishing', 'Published'];

// Calculate overall progress
const currentStageIndex = stages.indexOf(book.currentStage);
const stagesCompleted = currentStageIndex;
const progressInCurrentStage = book.stageProgress / 100;
const overallProgress = 
    ((stagesCompleted + progressInCurrentStage) / stages.length * 100)
    .toFixed(0);

// Function to generate the tracker HTML
const generateProgressTracker = () => {
    // Icon SVGs for each stage
    const icons = {
        'Prewriting': '<path d="M14.7 6.3..." />',
        'Drafting': '<path d="M11 4H4..." />',
        // ... etc
    };

    return `
        <div class="book-status-tracker">
            <!-- Header with percentage -->
            <div class="status-header">
                <span class="status-title">Writing Journey</span>
                <span class="status-percentage">${overallProgress}%</span>
            </div>
            
            <!-- Timeline -->
            <div class="status-timeline">
                <div class="status-progress-bar" 
                     style="width: ${overallProgress}%"></div>
                
                ${stages.map((stage, idx) => {
                    // Determine the state
                    let stepClass = '';
                    if (idx < currentStageIndex) {
                        stepClass = 'completed';
                    } else if (idx === currentStageIndex) {
                        stepClass = 'active';
                    } else if (stage === 'Published' && book.published) {
                        stepClass = 'published';
                    }
                    
                    return `
                    <div class="status-step ${stepClass}">
                        <div class="status-icon">
                            <svg viewBox="0 0 24 24">
                                ${icons[stage]}
                            </svg>
                        </div>
                        <span class="status-label">${stage}</span>
                    </div>
                    `;
                }).join('')}
            </div>
            
            <!-- Metrics -->
            <div class="status-details">
                <!-- Word count, stage, etc -->
            </div>
        </div>
    `;
};

Step 4: Insert into Book Card

STEP 4

In your book card template, add this line after the content warnings:

<!-- Content warnings section -->
${book.contentWarnings ? `<div class="content-warnings">...</div>` : ''}

<!-- ADD THIS LINE: -->
${typeof book.stageProgress === 'number' ? generateProgressTracker() : ''}

<!-- Rest of book info -->
💡 Why typeof book.stageProgress === 'number'?
This only shows the tracker if stageProgress exists and is a number. Books without progress data won't show the tracker.

Troubleshooting

Problem: Tracker doesn't appear

Check:

  • Is stageProgress defined as a number in your book data?
  • Did you copy all the CSS including .book-status-tracker?
  • Open browser console (F12) - any JavaScript errors?

Problem: Stages don't line up horizontally

Fix:

  • Check that .status-timeline has display: flex
  • Ensure width: 100% is set on the timeline
  • Make sure parent container isn't too narrow

Problem: Progress bar doesn't show

Fix:

  • Verify position: absolute on .status-progress-bar
  • Check that z-index: 1 is set (needs to be above background line)
  • Ensure width is being calculated correctly (check with browser inspector)

Quick Reference: Progress Calculation

// Example: Book is 30% through Drafting
currentStage = "Drafting"          // Stage 2 (index 1)
stageProgress = 30                 // 30% done with Drafting

// Calculation:
stagesCompleted = 1                // Prewriting is done
progressInStage = 0.3              // 30% as decimal
overallProgress = (1 + 0.3) / 6 * 100
                = 1.3 / 6 * 100
                = 21.67%           // Rounded to 22%

🎉 You're all set! Your progress tracker should now be working beautifully. As you update currentStage and stageProgress, the visualization will automatically update to show your writing journey.
